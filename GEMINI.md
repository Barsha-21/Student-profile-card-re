# Student Profile Card Project

## Overview
A simple React application that displays student profile cards.

## Getting Started
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`

## Project Structure
- `src/components/StudentCard.jsx`: Reusable card component.
- `src/App.jsx`: Main application container.
- `src/App.css`: Global styles and layout.
