# Student Profile Cards

A modern, component-driven React application developed with Vite. This project displays student information using reusable components, vibrant colors, and smooth CSS animations.

## Features
- **Component-Based:** Built using a modular `StudentCard` component.
- **Dynamic Styling:** Features custom CSS animations and hover effects.
- **Responsive Layout:** Flexbox-based grid for displaying student profiles.
- **Modern Tech Stack:** React 18+ and Vite 5+.

## Getting Started

### Prerequisites
- Node.js installed on your machine.

### Installation
1. Clone the repository.
2. Navigate to the project folder:
   ```bash
   cd Student-profile-card-re
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the development server:
```bash
npm run dev
```

### Production
Build the project for production:
```bash
npm run build
```
